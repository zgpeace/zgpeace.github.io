problem:
用hexo搭了个blog，之前一直正常，最近在deploy的时候无法连接到github了。
fatal: unable to access ‘https://github.com/xxxxx’: Empty reply from server.
请问这是怎么回事

solve:
翻墙即可

problem：
$hexo deploy
ERROR Deployer not found: github

The solution:
1: type change git
2: command npm install hexo-deployer-git –save
3: hexo d

it works
