  1. 1. Cocoapods 0.35.0 运行pod时候，有很多警告，解决方案：删掉所有cocoapods，然后重装，可以解决。
    1. 1.1. 1、看看目前有安装cocoapods相关程序：gem list –local | grep cocoapods
    2. 1.2. 2、删除相关cocoapods
      1. 1.2.1. sudo gem uninstall cocoapods -v 0.35.0
      2. 1.2.2. sudo gem uninstall cocoapods-core
      3. 1.2.3. sudo gem uninstall cocoapods-downloader
      4. 1.2.4. sudo gem uninstall cocoapods-plugins
      5. 1.2.5. sudo gem uninstall cocoapods-search
      6. 1.2.6. sudo gem uninstall cocoapods-trunk
      7. 1.2.7. sudo gem uninstall cocoapods-try
      8. 1.2.8. 查看是否还有没相关的： gem list –local | grep cocoapods
  2. 2. 运行 terminal in OS X 10.11 ， default enable rootless ,所以gem install “Operation not permitted - /usr/bin/pod”
    1. 2.1. sudo gem install cocoapods -v 0.35.0
    2. 2.2. 解决方法1: 命令增加 -n /usr/local/bin
    3. 2.3. 验证成功：pod –version

Cocoapods 0.35.0 运行pod时候，有很多警告，解决方案：删掉所有cocoapods，然后重装，可以解决。

1、看看目前有安装cocoapods相关程序：gem list –local | grep cocoapods

2、删除相关cocoapods

sudo gem uninstall cocoapods -v 0.35.0

sudo gem uninstall cocoapods-core

sudo gem uninstall cocoapods-downloader

sudo gem uninstall cocoapods-plugins

sudo gem uninstall cocoapods-search

sudo gem uninstall cocoapods-trunk

sudo gem uninstall cocoapods-try

查看是否还有没相关的： gem list –local | grep cocoapods

sudo gem install cocoapods -v 0.35.0

解决方法1: 命令增加 -n /usr/local/bin
